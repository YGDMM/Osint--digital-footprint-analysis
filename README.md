# OSINT Investigation Report – Digital Footprint Analysis
    OSINT investigation from an unknown image to identify a subject and analyze digital footprint
## Overview
This project presents an Open Source Intelligence (OSINT) investigation conducted with the objective of identifying an individual from a single image and analyzing their digital footprint.

The investigation demonstrates how publicly available information can be used to reconstruct an individual’s professional activity, digital infrastructure, and network of relationships.

## Objective
* Identify an individual from an unknown image
* Analyze their digital footprint using OSINT techniques
* Detect associated companies and infrastructure
* Establish relationships between entities and individuals

## Methodology

The investigation followed a structured OSINT approach:

* Reverse image search
* Source identification and validation
* Data extraction from public records
* Cross-referencing and correlation
* Technical infrastructure analysis (IP, ASN, hosting)

## Key Findings
* Successful identification of the subject through open sources
* Detection of multiple interconnected business entities
* Identification of associated digital infrastructure (ASN, IP, hosting services)
* Evidence of reduced online presence and digital footprint management
* Discovery of professional relationships within the same operational environment

## Tools Used

* Reverse image search engines
* OSINT tools (Hunter, Epieos, Truecaller)
* Public databases (RIPE, Whois)
* Web archives (Wayback Machine)

## Disclaimer

This report has been partially redacted to protect sensitive information and ensure compliance with privacy and ethical standards.
The objective of this project is to demonstrate OSINT capabilities, not to expose personal data.
